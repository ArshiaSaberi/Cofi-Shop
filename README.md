# ☕ Cofi Shop

یک پروژه فروشگاه کافی‌شاپ آنلاین که با استفاده از **Next.js**, **TypeScript**, **Tailwind CSS**  ساخته شده است.

---

🔗 [Cofi Shop](https://cofi-shop-z391.vercel.app/)

---

## 🛠️ تکنولوژی‌ها

- **Next.js**  
- **TypeScript** 
- **ReactQuery** 
- **Tailwind CSS**
- **ESLint**

## 🚀 شروع پروژه

ابتدا پروژه را کلون کنید:

```bash
git clone https://github.com/ArshiaSaberi/Cofi-Shop.git
cd Cofi-Shop




سپس بسته‌های مورد نیاز را نصب کنید:
npm install
# یا
yarn install
# یا
pnpm install





سرور توسعه را اجرا کنید:
npm run dev
# یا
yarn dev
# یا
pnpm dev
# یا
bun dev
